# 二分  
如果这个算法具有二段性，可以使用二分更快的找到满足性质的点（优于枚举查找）。  
时间复杂度：logn
```
//二分枚举最后一个能满足条件的 
int l=0,r=m;//有可能左边第一个也不能满足条件，l从0开始二分
while(l<r)
{
    int mid=l+r+1>>1;
    if(check(mid))	l=mid;//mid满足条件，答案在右区间（包含端点）
    else r=mid-1;//mid不满足条件，答案在左区间（不包含端点）
}
```


<font color='red'>注意数据范围是否需要开longlong</font>
