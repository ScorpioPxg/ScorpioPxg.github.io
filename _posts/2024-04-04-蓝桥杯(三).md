# 1. 快速幂
>可以快速进行乘方运算
>
>时间复杂度：O(logb)
>
>2024-04-04  


```
//a的b次对p取模
int qmi(int a,int b,int p)
{
	int res=1;
	while(b)
	{
		if(b&1)	res=(ll)res*a%p;
		a=(ll)a*a%p;
		b>>=1;
	}
	return res;
}
```

# 2. 最大公约数
>2024-04-04  

```
int gcd(int a,int b)
{
    return b ? gcd(b,a%b) : a; 
}
```
